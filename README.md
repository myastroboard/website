# web-myastroboard

Modern static showcase website for **myastroboard/myastroboard**.

## Vision

A premium, privacy-first, self-hosted product website that communicates:

- Open source by default
- Local-first data ownership
- No telemetry by default
- No account required
- No cloud lock-in

## Routes

- `/` landing showcase
- `/features/` product capabilities
- `/pricing/` philosophy pricing page
- `/docs/` documentation portal placeholder (future)

Future-compatible reserved areas:

- `/blog/` (future updates/devlogs)
- `/community/` (future contributor/community zone)

## Tech

- Pure static HTML/CSS/JS (no npm build stack required)
- Lightweight vanilla JS
- Custom CSS design system (dark-first, responsive, accessibility-minded)
- Directly deployable to OVH shared hosting via FTP/file upload

## Deployment

Upload the repository files as-is to your hosting root.

No Node.js, npm, SSR, database, or backend service is required.
