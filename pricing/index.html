What AGPL-3.0 means for you

The GNU Affero General Public License v3.0 is a free software license. It guarantees you four essential freedoms:

  • Freedom to use — Run MyAstroBoard for any purpose, in any environment.
  • Freedom to study — Read and understand the full source code at any time.
  • Freedom to modify — Adapt MyAstroBoard to your specific observatory setup.
  • Freedom to share — Distribute your modified version, as long as it remains AGPL-3.0.

The AGPL-3.0 adds one important requirement compared to GPL: if you run a modified version of MyAstroBoard over a network (e.g., shared with your astronomy club), you must make the source code of your modifications available.

This ensures that the community always benefits from improvements. It also means no corporation can take MyAstroBoard, add proprietary features, and offer it as a closed service without contributing back.

Transparent by design

You are the administrator of your own instance. There is no hidden data monetization, no mandatory account layer, and no cloud dependency required to keep your observatory dashboard online.

  • All astronomical calculations run locally on your hardware — no external API call for SkyTonight.
  • Weather data fetched from Open-Meteo, which is itself an open-source project with no API key required.
  • User data is stored only in your Docker volume and never sent anywhere.

If you find MyAstroBoard useful, the best way to support the project is to star the repository on GitHub, report bugs, contribute translations, or submit feature requests via GitHub Issues.
